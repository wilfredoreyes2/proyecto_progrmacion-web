Este es un proyecto que se basa en reproducir trailer y que contiene un crud que solo puede entrar el administrador, 
lo decidimos llamar 'MOVIEPEDI'.

Está hecho con HTML, CSS, JS, PHP enlazado a una base de datos de PHP MY ADMIN.

Para que pueda funcionar:

1. Debe tener descargado el servidor local XAMPP, y dentro del disco local, localizar XAMPP,
luego la carpeta 'htdocs', y dentro de esta incluir esta carpeta/proyecto llamado 'pagina de trailers1'. 

2. Encender el servidor. 

3. Dentro de su navedador copiar o escribir esta ruta:'http://localhost/pagina%20de%20trailers1/' y listo,
va a cargar el proyecto.
